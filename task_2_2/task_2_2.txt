Клиент для pastebin (или похожего сервиса). Реализовать поддержку
следующих функций:
- создание новой текстовой записи из stdin или файла, указанного через
аргументы командной строки, автоматического (на основе расширения файла)
или ручного (через аргументы командной строки) выбора языка подсветки;
- создание новой текстовой записи должно возвращать id;
- вывод на консоль или в файл (в зависимости от аргументов) текстовой
записи по id;
- для создания и скачивания должен использоваться один и тот же скрипт.

Пример использования:
Создать новую запись с типом текст: echo testing | ./pastebin.sh -t text
Вывести на экран запись по id=QEXe0syg: ./pastebin.sh QEXe0syg
